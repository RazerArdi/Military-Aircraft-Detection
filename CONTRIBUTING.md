# Contributing to ARS Command Center

Thank you for your interest in contributing to the Aerial Reconnaissance System!

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone the project** to your own machine.
3.  **Commit changes** to your own branch.
4.  **Push** your work back up to your fork.
5.  Submit a **Pull Request** so that we can review your changes.

## Reporting Issues

If you find a bug or have a suggestion, please open an Issue on GitHub and describe:
- Steps to reproduce the bug.
- Expected behavior.
- Screenshots (if applicable).

## Code Style

- We follow standard Python **PEP 8** guidelines.
- Please ensure your code is commented and documented.
